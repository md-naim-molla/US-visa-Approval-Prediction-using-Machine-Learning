# US-visa-Approval-Prediction-using-Machine-Learning

## How to run?

```bash
git clone https://github.com/md-naim-molla/US-visa-Approval-Prediction-using-Machine-Learning.git
```

```bash
conda create -n visa python=3.8 -y
```

```bash
conda activate visa
```

```bash
pip install -r requirements.txt
```
